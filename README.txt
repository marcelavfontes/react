React course commands
- [ ] install yarn (node 8 or higher)
	npm install -g yarn (-g install globally)

- [ ] install light server
yarn -> yarn global add live-server
node -> npm install -g live-server

- [ ] Running the server 
	live-server /folder/

- [ ] Babel - compiler that converts jsx to js
needs preset to compile, it doesnâ€™t do it by default
yarn -> yarn global add babel-cli@6.24.1
node -> npm -g babel-cli@6.24.1

- [ ] install env and react preset
yarn init -> set project to use yarn to specify local dependencies. Generates the package.json file

yarn add babel-preset-react@6.24.1 babel-preset-env@1.5.2
-> generates node_modules and yarn.lock files

src/app.js -> contains the jsx code
scripts/app.js -> contains the autogenerated file (babel transformations)

- [ ] Babel command to compile
babel src/app.js --out-file=public/scripts/app.js --presets=env,react --watch 
-> compile with babel, overwrites the scripts file (--watch to autodetect the changes in the file)
